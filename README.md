**How to Calculate Age in Excel**
Many professionals need to know how to calculate age in Excel. It is a common task for HR and data management. There are several methods.

**Using the DATEDIF Function**
The best way to calculate age from date of birth in excel is with the DATEDIF function. This function calculates the difference between two dates.

**Formula:**
=DATEDIF(A1, TODAY(), "Y")

In this formula, A1 is the cell with the birth date. TODAY() provides the current date. "Y" returns the full years of age. To get months and days, use "YM" and "MD" in separate formulas.

**Using Simple Year Subtraction**
A simpler formula is =(TODAY()-A1)/365. However, it is less precise. It does not account for leap years perfectly. For formal use, the DATEDIF function is superior.
